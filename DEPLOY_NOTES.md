> 마지막 배포: `f2ddfd8` 이후 dev에 쌓인 것들. "배포"라고 하면 dev→master 머지+푸시.
> 적용 시: **봇 재시작** 필요. 확장 변경분은 **크롬 확장 새로고침 + OTT 탭 F5** 필요. (플러그인은 이번 변경 없음 → ST 재시작 불필요)

## 1. 사람답게 (타이핑 감지 배칭)
- 유저가 **입력 중이면**(Discord TypingStart) 답을 미뤘다가, 멈추면 **한 번에** 답함. 여러 줄 텀 두고 쳐도 안 끊김.
- 갠톡 + 단톡 둘 다 적용. 최대 대기 25초 캡(`replyMaxWaitMs`), 타이핑 인식 윈도우 ~6초(`typingGraceMs`).
- 단톡: 저장/페르소나 프록시는 즉시, 생성만 배칭.

## 2. 답변 자연스러움
- **갠톡 줄수 가변**: 늘 3줄 고정 → 1줄/2줄/버스트 들쭉날쭉.
- **단톡 자유 티키타카**: 로스터 순서대로 한 번씩(A→B→C) 돌던 것 → 같은 사람 여러 번, 순서 자유.
- **단톡 유저 발화 반응**: 지들끼리만 떠들지 않게, 최소 한 명은 유저에게 반응(즉답 강제는 아님, 자연스럽게).

## 3. 영화 같이보기 개선
- **쿠팡플레이 추가** (유튜브/넷플/디즈니+/쿠팡플레이).
- **자막 캡처**: 표준 TextTrack(activeCues) 1순위 + 비디오 하단 텍스트 자동탐지 폴백 (사이트 DOM 몰라도 잡기 시도).
- **영화 이름 수동 입력칸** (자동감지 실패 시).
- **옆에서 같이 보는 느낌**: 자막 반응만 말고 유저에게 말 걸기/잡담, 대화 히스토리 반영.
- **영상 맥락 인지**: 영화 채널의 모든 답변(유저가 말 걸 때 포함)에 "지금 보는 중 + 최근 자막" 주입.
- **단톡 영화**: 볼 사람만 참여(전원 강제 X).
- **반응 지연 수정**: 묵은 자막 백로그 버리고 최근 6줄만 반응 + 간격 60→35초(`movieReactSec`).

## 4. NPC 단톡 (`/npc`)
- 확장에서 채널에 "NPC그룹" 체크 + NPC 로스터 입력 → 그 채널은 **1:1 유지**.
- 그 채널에서 `/npc create` → 로스터를 복사한 **파생 단톡 채널** 자동 생성 (npcgroups.json에 봇이 저장).
- 갠톡↔NPC단톡 기억/상태 공유, NPC 선톡(가끔 메인 몰카 사진), `/npc add/remove/list/delete`.

## 6. 통화 목소리 = 음성메모 목소리 (클론)
- **클론 목소리(`voiceEngine: "clone"` + 채널/전역 `rvcVoice`)가 설정된 채널은 `/call`도 그 목소리로** — 음성메모와 통화의 목소리 통일. edge-tts(낭독체) 탈출.
- 파이프라인: 유저 발화 → Gemini STT → 응답 생성(flash) → **문장 단위로 클론 합성·재생** (첫 문장 준비되는 즉시 발화 시작, 다음 문장은 재생 중 합성).
- **추임새 선재생**: 통화 시작 시 "Mmm...", "Hmm?" 등을 캐릭터 목소리로 미리 합성해뒀다가, 유저 말이 끝나면 즉시 재생 → 합성 지연(2~3초)을 "생각하는 시간"으로 위장.
- 말 끊고 들어가기(barge-in): 유저가 말 시작하면 재생 중지 + 대기 문장 폐기(`speakGen`).
- 통화 언어는 음성메모와 동일하게 영어 고정(목소리 모델이 영어) — 유저는 한국어로 말해도 알아들음. `callLanguage`로 강제 변경 가능.
- 클론 합성 실패 시 edge-tts 폴백. 통화 시작 시 클론 서버 워밍업(콜드스타트 대비).
- 신규 config 키: `callEngine` (""=자동, "live"=클론 무시하고 Live 강제), `callFillers` (추임새 문구 배열, 기본 내장).
- 변경 파일: `src/bot.js`, `src/voice-call.js`, `config.example.json`

## 5. 음성 통화 (`/call`) — 무료 스택
- 디코 **음성채널**에서 캐릭터와 실시간 통화. 유저가 음성채널 들어가서 갠톡 채널에서 `/call start`.
- STT: **Gemini**(이미지 프로필 또는 Gemini 채팅 프로필 키 재활용) / TTS: **edge-tts**(무료, MS Edge 보이스).
- 말 끊고 들어가기(barge-in), 통화 내용 갠톡 히스토리 저장(📞 prefix) + 텍스트 채널에 자막 표시.
- 유저가 음성채널 나가면 자동 종료. `/call end`로도 끊기.
- ⚠️ **이번엔 서버에서 `npm install` 필요** (@discordjs/voice, msedge-tts 등 신규 의존성).
- 신규 config 키: `callVoice` (기본 ko-KR-InJoonNeural), `callResponseTokens` (기본 512), 채널별 `channels[id].callVoice`.
